# 文档索引

[简体中文](README.md) | [English](README_en.md)

本目录属于 `minillm-from-scratch` 项目，Git 根目录是本文档的上一级目录。

## 项目文档

- [`../README.md`](../README.md)：项目介绍、环境、数据、训练、评测和演示命令。
- [`tutorial/README.md`](tutorial/README.md)：面向初学者的七章 notebook 学习路线。
- [`tutorial/notebooks/`](tutorial/notebooks/)：融合公式、解释、代码与训练命令的完整教程。
- [`tutorial/README_en.md`](tutorial/README_en.md)：English beginner course index。
- [`tutorial/notebooks_en/`](tutorial/notebooks_en/)：英文七章教程，与中文版本共享代码单元。
- [`theory.md`](theory.md)：核心公式与算法速查。
- [`experiment_report.md`](experiment_report.md)：正式实验设置、实测结果、结论和限制。

## 论文资料

- [`../papers/references_and_analysis.md`](../papers/references_and_analysis.md)：论文目录、设计决策映射和 BibTeX。
- `../papers/*.pdf`：本地论文原文，只作离线参考；该目录被 Git 忽略，不随代码仓库提交。

## 本地实验产物

- `../results/`：统一评测报告、JSON/CSV 指标和生成样本。
- `../logs/`：训练及评测日志。
- `../checkpoints/`：模型权重。

以上实验产物和论文 PDF 默认不进入 Git；GitHub 仓库跟踪代码、配置、小型数据、实验报告和可执行教程。
